# 🔥 SatAlert

## Sistema de Monitoramento de Queimadas e Desmatamento via Satélite

Projeto desenvolvido para a **Global Solution FIAP 2026**, com o objetivo de utilizar dados públicos obtidos por satélites para identificar focos de queimadas e áreas de desmatamento, gerando alertas em tempo quase real para produtores rurais, órgãos ambientais e instituições responsáveis pela preservação ambiental.

---

# 📖 Sobre o Projeto

O **SatAlert** é uma plataforma inteligente que integra informações provenientes de APIs públicas disponibilizadas pela **NASA** e pela **Agência Espacial Europeia (ESA)**, realizando o processamento automático desses dados para auxiliar no monitoramento ambiental.

A solução centraliza informações de diferentes fontes em uma única plataforma, oferecendo uma visualização simples e intuitiva através de um **Dashboard Web** e de um **Aplicativo Mobile**, permitindo que produtores rurais e órgãos ambientais acompanhem possíveis ocorrências de queimadas e desmatamento em tempo quase real.

---

# 🎯 Objetivos

- 🌎 Monitorar focos de queimadas utilizando dados de satélites;
- 🌳 Detectar possíveis áreas de desmatamento;
- 🚨 Gerar alertas automáticos em tempo quase real;
- 🗺️ Disponibilizar mapas e indicadores ambientais;
- 👨‍🌾 Auxiliar produtores rurais e órgãos responsáveis na tomada de decisão;
- ♻️ Incentivar ações preventivas para preservação ambiental.

---

# 🌎 Problema

Atualmente, grande parte dos dados ambientais está distribuída em diferentes plataformas e exige conhecimento técnico para consulta e interpretação.

Essa descentralização dificulta respostas rápidas e ações preventivas, aumentando os impactos ambientais causados por queimadas e desmatamento ilegal.

O **SatAlert** busca resolver esse problema reunindo todas essas informações em uma plataforma única e acessível.

---

# 💡 Solução Proposta

A plataforma realiza:

- 📡 Consumo de APIs públicas de satélites;
- 🧠 Processamento inteligente das informações coletadas;
- 🔥 Identificação automática de focos de queimadas;
- 🌲 Detecção de alterações relevantes na cobertura vegetal;
- 🚨 Geração de alertas para usuários cadastrados;
- 📊 Visualização das ocorrências em mapas e dashboards;
- 📝 Consulta ao histórico de eventos monitorados.
